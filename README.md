# Zoom Video Conferencing Web App
A real-time video conferencing application
that allows users to create and join meeting rooms, 
communicate via video/audio, and collaborate online.
---
## Features
- Create and join meeting rooms
- Real-time video & audio communication
- Screen sharing
- Live chat
- Multiple participants support
---
## Tech Stack
Frontend
- React.js
- HTML5
- CSS3
- JavaScript
Backend
- Node.js
- Express.js
Real-time Communication
- WebRTC
- Socket.io
Database
- MongoDB
---
## Installation
1. Clone the repository
 git clone
https://github.com/Anjali-yadav513/Apna_Videocall.git

2. Move into project folder
cd  Apna_Videocall
3. Install dependencies
npm install
4. Start the server
npm start

---
## Usage
1. Open the application in your browser
2. Create a meeting room
3. Share the meeting link
4. join and start video conferencing
